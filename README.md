# Learn Chinese 3000

Mandarin Flashcards is a web application that helps users 
learn Mandarin Chinese vocabulary using a flashcard system. 
Users can navigate through the cards, add new cards, delete cards, 
and upload CSV files with vocabulary data.

## Features

- Randomly generated flashcards
- Upload CSV files with card data
- Add and delete cards
- Pagination (not yet implemented)

## Usage

1. Visit http://localhost:4990 in your web browser to access the application.
2. Use the arrow buttons to navigate through the cards.
3. Click the "+" button to add a new card.
4. Click the "Upload CSV" button to upload a CSV file with vocabulary data.

## CSV Format

The CSV file should have the following columns:

- word
- translation
- pinyin
- type
- category
